# RSS Feed for [rss.nickmonaco.me](rss.nickmonaco.me)

Server: [Serverless API using Apollo Server + Mongoose](https://github.com/monaco89/nickmonaco-rss-api)

## Libraries

- React
- Apollo Client, GraphQL
- Formik
- React Testing Library
- React Suite

## Run locally

1. `yarn install`
2. `yarn start`
